## Run Locally

Clone the project

```bash
  git clone https://github.com/ArnavK-09/simplisticchatbotever.git
```

Go to the project directory

```bash
  cd my-project
```

Install all dependencies

```bash
  npm install
```

Start the server

```bash
  npm run 
 ```
  
## Tech Stack
Using **Express, CSS 3, HTML 5, PUG, Javascript**
